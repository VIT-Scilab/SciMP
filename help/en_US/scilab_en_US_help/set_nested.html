set_nested

Enables nested parallelism.

Syntax

set_nested(b)

Arguments

b

Input argument (A nonzero value enables nested parallelism, while zero disables nested parallelism)

No Output argument

Description

This function is used to enable or disable nested parallelism.

This function in turn calls the omp_set_nested in C.

Examples

set_nested(1)
