critical

To enable the critical region.

Syntax

critical(int a)

Arguments

a

The input can be 1 or 0 to enable or disable the critical region respectively

-

No output argument

Description

This function is used to enable the critical region.

This function inturn executes the function under it in #pragma omp critical region in C.

Examples

critical(1)
