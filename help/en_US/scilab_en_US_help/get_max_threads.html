get_max_threads

Return the maximum number of threads used for the current parallel region that does not use the clause num_threads.

Syntax

a = get_max_threads(void)

Arguments

No input arguments needed.

a

Output argument (Maximum number of threads that can be used to form a process if a parallel region without num_threads clause is encountered)

Description

This function returns the maximum number of threads that can execute concurrently in a single parallel region

This function in turn calls the omp_get_max_threads in C.

Examples

a = get_max_threads()
