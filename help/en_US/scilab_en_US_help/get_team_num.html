get_team_num

Returns the team number of the calling thread.

Syntax

a = get_team_num(void)

Arguments

No input arguments needed.

a

Output argument (Team Number Of Calling Thread whose range is between 0 and get_num_teams - 1)

Description

This function returns team number or process number of calling thread

This function in turn calls the omp_get_team_num function of omp library in C.

Examples

a = get_team_num()
