get_total_thread

Get the total number of threads running in the parallel region.

Syntax

a = get_total_thread(Void)

Arguments

Void

No arguments needed.

a

Output argument (returns total number of threads)

Description

This function returns the total number of threads running in the parallel region.

This function inturn calls the omp_get_num_threads in C.

Examples

a=get_total_thread()
