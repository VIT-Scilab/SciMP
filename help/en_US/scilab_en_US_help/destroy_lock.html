destroy_lock

Uninitializes a lock.

Syntax

destroy_lock(void)

Arguments

No input and output arguments needed.

Description

This function destroys simple lock.

In order to be destroyed, a simple lock must be in the unlocked state.

This function in turn calls the omp_destroy_lock function of omp library in C.

Examples

destroy_lock()
