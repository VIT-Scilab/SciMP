mat_mul

Returns the product of 2 matrices.

Syntax

C = mat_mul(A,row1,col1,B,row2,col2)

Arguments

A

First Matrix

row1

No. of rows in Matrix A

col1

No. of cols in Matrix A

B

Second Matrix

row2

No. of rows in Matrix B

col2

No. of cols in Matrix B

C

Output argument (result of the matrix multiplication of A and B

Description

This function returns the product of 2 matrices. The no. of cols of first matrix should be equal to the no. of rows of second matrix

This function in turn calls a C function where the multiplication is carried out using multiple threads.

Examples

a = [1,2; 3,4; 5,6]
b = [1, 2, 3, 4; 5, 6, 7, 8]
c = mat_mul(a,3,2,b,2,4)
