set_dynamic

Enable or disable the dynamic adjustment of the number of threads within a process. The function takes the language-specific equivalent of true and false, where true enables dynamic adjustment of team sizes and false disables it.

Syntax

set_dynamic(b)

Arguments

b

Input argument (A nonzero value means threads will be dynamically adjusted.)

No output arguments needed.

Description

This function enables or disables dynamic adjustment of number of threads within a process. By default, it is disabled

This function in turn calls the omp_set_dynamic function of omp library in C.

Examples

set_dynamic(1)
