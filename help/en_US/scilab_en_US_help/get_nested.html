get_nested

This function returns a value that indicates if nested parallelism is enabled.

Syntax

a = get_nested(void)

Arguments

No input arguments needed.

a

Output argument (A nonzero value means nested parallelism is enabled.)

Description

This function returns a value which indicates if nested parallelismis enabled or not.

This function in turn calls the omp_get_nested function of omp library in C.

Examples

a = get_nested()
