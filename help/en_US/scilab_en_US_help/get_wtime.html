get_wtime

Get the elapsed wall clock time.

Syntax

a = get_wtime(void)

Arguments

No input arguments needed.

a

Output argument (returns the number of seconds elapsed)

Description

This function returns the number of seconds elapsed from some time.

This function inturn calls the omp_get_wtime in C.

Examples

a = get_wtime()
