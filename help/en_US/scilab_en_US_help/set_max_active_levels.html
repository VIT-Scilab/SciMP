set_max_active_levels

Limits the maximum allowed number of nested, active parallel regions

Syntax

set_max_active_levels(b)

Arguments

b

Input argument (Number of active and nested parallel regions)

No Output argument

Description

This function limits the maximum allowed number of nested, active parallel regions.

This function in turn calls the omp_set_max_active_levels in C.

Examples

set_max_active_levels(5)
